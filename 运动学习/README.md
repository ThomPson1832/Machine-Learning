# 运动分析软件

## 项目简介
基于普通摄像头的无标记3D运动分析软件，结合AI技术实现人体关键关节点识别和运动数据分析。

## 功能特性
- 基于普通摄像头的无标记3D运动分析
- AI姿态检测与人体关节点识别
- 角度、速度、加速度等运动参数计算
- 3D运动数据可视化
- 实时视频流处理

## 技术栈
- Python 3.8+
- OpenCV (视频处理)
- MediaPipe (姿态检测)
- PyTorch/TensorFlow (深度学习)
- Three.js/PyOpenGL (3D可视化)
- Tkinter/PyQt (GUI界面)

## 安装依赖
```bash
pip install -r requirements.txt
```

## 运行程序
```bash
python main.py
```
